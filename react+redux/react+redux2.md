主要使用的技术有react+redux+immutable+weui+node+spring。

在一开始使用react+redux的过程中并没有发现一个细节，就是在reduver的时候用到了 `immutable` 这个工具包。这个包和react搭配简直爽到爆，去除了很多烦恼。

工作中发现，这些个前端框架果然都是熟练型的，不管是angular react 基本都能在了解api后1周内熟练上手，我想着也就是开发框架的原因之一了。但是话不能说的太满，这些个前端框架在实际开发过程中还有一部分方法并没有用到过。能用到的现在也就是一些皮毛。由于之前遇到的问题都已经解决且时间比较长了，所以现在没办法再写出遇到的问题了。

使用react，最重要关心的就是`构建组件`。组件要有`封装性`，且能对`复用`，`测试`和`分离`都最大程度的支持。各自组件有各自的状态，当状态发生改变时，会重新刷新组件改变内容。
